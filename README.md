# ShenYin

本地数据整理工具，纯离线运行。

## 功能

- 支持纯文本和任意文件的双向处理
- 支持访问码模式与凭据模式
- 文本输出使用高效编码，无外部格式标记
- 支持多种压缩等级
- 本地凭据管理（生成 / 导入 / 下载 / 删除）
- 默认打开本地浏览器页面，所有数据仅在本机流转

## 运行

```powershell
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
.\.venv\Scripts\python app.py
```

启动后自动打开：

```text
http://127.0.0.1:8765
```

也可以直接下载 Release 中的便携版 `local-workspace.exe`，双击即可使用，无需安装。

## 使用建议

- 简单场景：直接使用访问码模式
- 多设备或多人协作：先在凭据管理中生成或导入凭据，再使用凭据模式
- 文本传输：保持文本格式输出，结果可直接复制粘贴
- 文件传输：选择文件格式输出，体积更小

## 发布

- 推送到 `main` 分支会自动构建并发布滚动预发布版
- 打 `v*` 标签（如 `v1.0.0`）会发布正式版本
